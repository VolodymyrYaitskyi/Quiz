This app is using json-server 
To use this app you should do next actions : 
1)aware that Node.js installed on your pc
2)type command "npm init -y" this will create file package.json
3)Install JSON Server using command npm install -g json-server
4)In package.json create   "scripts": {
    "start": "json-server --watch ./server/questions.json"
  }
4) type command "npm start" to load json-server
